## 📅 Weeks 1–6 · Foundations (Python + Math)

- [ ] **Week 1–2** — Python syntax: variables, types, loops, conditionals, functions
- [ ] **Week 3–4** — Data structures: lists, dicts, sets, tuples, comprehensions, file I/O
- [ ] **Week 5** — Functions deep-dive, error handling, modules; **Git + GitHub basics**
- [ ] **Week 6** — Descriptive stats + linear algebra intuition (vectors/matrices)

**Resources**
- [ ] freeCodeCamp — Scientific Computing with Python → https://www.freecodecamp.org/learn/scientific-computing-with-python/
